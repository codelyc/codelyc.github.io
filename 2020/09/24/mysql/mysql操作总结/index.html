非空+唯一约束 z主键

第一个被定义为非空+唯一的那一列会成为这张表的primary key
一张表只能有一个主键
也可以通过primary key定义一个主键

自增

只能对数字有效，自带非空约束.
至少是unique的约束之后才能使用auto_increament 所以它直接变成了主键所以我们常用写法.

分组

参考博客
group by 根据谁分组,可以求这个组的总人数,最大值,最小值,平均值,求和 但是这个求出来的值只是和分组的字段对应 ，并不和其他任何字段和分组后的字段数据关联起来.

聚合函数

count 求个数
max 求最大值
min 求最小值
sum 求和
avg 求平均

having 过滤语句

在having条件中可以使用聚合函数（包含group by语句）,在where中不行
适合去筛选符合条件的某一组数据,而不是某一行数据
先分组再过滤 : 求平均薪资大于xx的部门,求人数大于xx的性别,求大于xx人的年龄段
