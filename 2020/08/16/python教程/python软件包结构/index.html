python 软件开发规范

bin

存放执行脚本

conf

表示为配置文件

比如：用到了某一个值， 这个值在程序执行的过程中会被修改 ， 且一旦修改,要修改代码，且要修改多处，这样的值，我们应该单独的写在conf文件夹下面的一个文件中。 来保证以后所有的程序修改这个值,都需要从这个文件中读取，在需要修改这个值的时候只需要修改配置文件中的一处就可以了

code

存放核心逻辑

db

存放数据库文件

lib

存放自定义的模块与包，不是内置模块，也不是第三方模块（可以从网上下载下来的模块）
